# Executive Tableau Dashboard

Business Problem

The objective of this project is to develop an executive Tableau dashboard that enables retail leadership to monitor sales performance, profitability, customer segments, shipping efficiency, discount impact, and return patterns for better business decision-making.

Dataset Description

The dataset contains retail sales transactions including order details, customer information, product categories, sales, profit, discounts, shipping details, delivery days, customer ratings, and return information.

Tableau Workbook

File: "tableau/executive_dashboard.twbx"

Calculated Fields

- Profit Margin
- Cost
- Average Order Value
- Return Rate
- Shipping Delay Bucket

Dashboard Components

- Total Sales KPI
- Total Profit KPI
- Profit Margin KPI
- Sales Trend
- Regional Performance
- Category Profitability
- Customer Segment
- Shipping Performance
- Discount vs Profit
- Return Analysis

Filters and Interactions

- Region
- Category
- Customer Segment
- Ship Mode
- Interactive dashboard filter action

Key Business Insights

- Strong regional sales differences.
- Technology products generate higher profits.
- Corporate customers contribute significant sales.
- High discounts reduce profitability.
- Shipping efficiency varies by shipping mode.
- Return patterns identify potential quality issues.

Dashboard Story Summary

The dashboard provides leadership with an integrated view of sales, profitability, customer behaviour, logistics, and product performance to support strategic decision-making.

Assumptions and Limitations

- Calculated fields were derived using available dataset columns.
- Historical data only; no forecasting included.
- Results depend on data quality and completeness.
